## 🎯 用户使用方式

### 方法1：npm直接安装（推荐）
```bash
npm install -g wxcloud-mcp
```

### 方法2：从GitHub安装
```bash
npm install -g github:iptton-ai/wxcloud-mcp
```

### 方法3：克隆构建
```bash
git clone https://github.com/iptton-ai/wxcloud-mcp.git
cd wxcloud-mcp
./install.sh
```

### Claude Desktop配置
```json
{
  "mcpServers": {
    "wxcloud": {
      "command": "wxcloud-mcp"
    }
  }
}
```
